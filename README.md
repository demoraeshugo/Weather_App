## Purpose
- Weather app that uses a 3rd party API to display current and future weather data based on user location.

## Goals
- Clean scalable componenets
- API usage
- Room for expansion 


## Component Hierarchy
- App
  - Header
  - Body
    - Location
    - Current
    - Forecast
